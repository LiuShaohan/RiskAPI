# Risk API

### 全局字段说明

userId：为业务系统向风控系统传输时，用来标示唯一用户。

submitTime：为业务系统向风控系统提交风控数据时的时间。

### 补充说明

1. 运营商认证，尽可能使用魔蝎数据定义的接口与参数。
2. 风控报告字段，待接好三方数据源及风控策略集后，根据情况再产出对应的接口文档。
